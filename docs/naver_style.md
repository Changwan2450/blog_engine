# Naver Blog Writing Style

이 문서는 네이버 블로그 글쓰기 스타일 규칙이다.

## Title

- 26-34 characters
- Include one main keyword
- Example: "AI 에이전트 자동화 실험: 진짜 생산성이 올라갈까?"

## Intro

- 3-4 short sentences
- Explain:
  - what the topic is
  - why it matters

## Structure

- H2 sections:
  - 핵심 요약
  - 왜 중요한가
  - 바로 적용하기

## Ending

- Checklist format
- Example:

### 오늘 핵심 정리

- AI 자동화는 반복 작업에 가장 강하다
- 작은 실험부터 시작하는 것이 중요하다
- 데이터 기반으로 개선해야 한다
